========================================
📤 分发方式
========================================

方式 1: 网盘分享
- 将分发包压缩成 ZIP 或 RAR
- 上传到百度网盘/阿里云盘/OneDrive
- 分享链接给用户

方式 2: Docker Hub
- docker tag ai-travel-planner:latest 用户名/ai-travel-planner
- docker push 用户名/ai-travel-planner:latest
- 用户直接 pull 使用

方式 3: GitHub Release
- 创建 Release
- 上传压缩包（注意 2GB 限制）
- 提供下载链接
